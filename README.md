# Sentence Tracker Application

This repository contains my **Python programs** for the coursework tasks.

---

## Task: Sentence Structure Application

This program takes a sentence as input from the user and performs the following actions:

- Counts the **number of words** in the sentence.
- Displays **each word** in a list format.
- Identifies and displays the **longest word**.
- Counts the **number of vowels and consonants** in the sentence.

### Example Output

Enter the sentence: I love cooking

Sentence: I love cooking

Word 1: I

Word 2: love

Word 3: cooking

Longest word: cooking

Number of vowels: 6

Number of consonants: 6

Number of words: 3
